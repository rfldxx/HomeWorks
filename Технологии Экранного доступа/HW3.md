### Ход работы:
1.  Научимся более продвинутой навигации, чем просто `tab`. У NVDA есть [Объектная навигация](https://download.nvaccess.org/releases/stable/documentation/ru/keyCommands.html#ObjectNavigation):

2. Нам же понадобиться следующие команды NVDA (используется ноутбучная раскладка):
    - Перейти на уровень выше: `NVDA+Shift+↑`
    - Перейти на уровень ниже: `NVDA+Shift+↓`
    - Итерация по элементам на текущем уровене: `NVDA+Shift+←` или `NVDA+Shift+→`
    - Поставить указатель мыши к текущему элементу (нам это будет нужно для последующего нажатия по этому элементу): `NVDA+Shift+M`
    - Нажатие ЛКМ: `NVDA+[`
    - Нажатие ПКМ: `NVDA+]`

3. Также вполне полезные следующие комбинации в Windows:
    - перейти на рабочий стол: `Win+D`
    - перейти к панели задач: `Win+T`
    - перейти к значкам на панели задач: `Win+B`
    - контекстное меню: `Shift+f10`

4. Перехожу в браузер. Чтобы перейти сразу в строку поиска нажимаю: `Ctrl+L`. Для отправки письма буду использовать Gmail.

5. В Gmail очень удобным оказалось, что можно обойтись только табом для обхода нужных элементов. Табом дошёл до кнопки написать письмо. Далее, при наборе письма, `tab` позволяет переключать поля редактирования! То есть не надо пользоваться объектной навигацией чтобы выйти из области набора текста и перейти к другим элементам. Это позволило средствами из первых домашек прикрепить нужный документ к письму.
