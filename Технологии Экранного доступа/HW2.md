## Домашнее задание № 2
Создать документ MS Word. Документ должен содержать: титульный лист; оглавление; введение, две главы, заключение. \
Параметры документа: шрифт times new roman 14, междустрочный интервал 1,5 строки. \
Заголовки выравниваются по центру, основной текст по ширине. \
Текст для заполнения структуры документа выбирается на ваше усмотрение. \
Во введении глаголы нужно сделать курсивом, в первой главе полужирным шрифтом, во второй главе раскрасить красным цветом, в заключении раскрасить зеленым.

### Ход работы:
1. У меня нет на ноутбуке MS Word, поэтому буду использовать браузерную версию GoogleDocs

2. Самая важная комбинация: поиск по меню: `Alt+/`
    - при поиске появляется список, по которому можно перемещаться стрелками вверх / вниз
    - NVDA озвучивает какой сейчас пункт и из скольки
    - также NVDA озвучивает комбинации клавиш (если они есть) для выбранного поля
    - кончено не без минусов: это не такая мощная вещь как поисковик, нужно набирать почти слово в слово нужный инструмент

3. А можно ли выйти из поля редактирования и походить по другим элементам страницы?
    - я нашёл только слегка обходной путь
    - перейти к формату текста: `Alt+O` (это в принципе полезная комбинация, там можно выбрать междустрочный интервал)
    - а затем табаться
    - некоторые объекты имеют и вложенную структуру (по ним можно перемещаться стрелками, например: "Основной панель инструментов")
    - если хочется быстрее вернуться к самому тексту, то можно просто перезагрузить страницу и курсор окажется в начале текста (а, как будто иногда и просто `esc` работает)

4. В итоге для форматирования выделяем нужный текст, жмём `Alt+/` (поиск по меню), пишем нужную настройку. \
В случае с шрифтом и цветом, можно в "поисковике" писать сразу нужный вариант
